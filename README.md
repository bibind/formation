# Formation "API Platform Avance – Projet Marketplace"

Ce depot contient l'ensemble des livrables pedagogiques et un squelette de projet pour la formation.

## Structure

- `deliverables/00_admin/` : hypotheses, conventions, audit interne
- `deliverables/01_programme/` : programme complet, objectifs, planning
- `deliverables/02_support_formateur/` : guides et supports formateur
- `deliverables/03_support_stagiaire/` : supports apprenants
- `deliverables/04_ateliers/` : ateliers progressifs
- `deliverables/05_corriges/` : corriges techniques
- `deliverables/06_evaluations/` : matrices, grilles, QCM, etude de cas
- `project/` : documentation technique et squelettes de classes

## Conventions

Tous les livrables sont en Markdown et structurés avec H1/H2/H3. Les contenus incluent objectifs, contexte, contraintes, criteres de reussite, pieges frequents, preuves audit possibles et temps estime par atelier.

## Onboarding

- `make install`
- `make test`
- `make ci`
- `make reset-all`
